Contrast-O-Vision requires support for SVG Filter Effects on HTML content, which means it won't work in...

  • Internet Explorer
  • Safari (due to a bug)
  • ...or whatever you're using.

It does work in other browsers though! Honest!

Firefox

Firefox's rendering tends to be more accurate, but can be slower on some devices.

Firefox for desktops & laptops Firefox for Android

Chrome

Chrome doesn't render Contrast-O-Vision perfectly, but it can have a smoother frame rate if your device is struggling.

Chrome for desktops & laptops Chrome for Android
Edge

Still mostly untested. I haven't heard of any bugs, anyway.

You'll need Windows 10 or newer. If you have it already, Edge should already be installed.
